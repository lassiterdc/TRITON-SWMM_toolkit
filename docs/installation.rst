Installation
============

Stable release
--------------

To install TRITON-SWMM Toolkit, run this command in your terminal::

    uv add TRITON-SWMM_toolkit

Or if you prefer to use ``pip``::

    pip install TRITON-SWMM_toolkit


From source
-----------

The source files for TRITON-SWMM Toolkit can be downloaded from the
`GitHub repo <https://github.com/lassiterdc/TRITON_SWMM_toolkit>`_.

You can either clone the public repository::

    git clone git://github.com/lassiterdc/TRITON_SWMM_toolkit

Or download the
`tarball <https://github.com/lassiterdc/TRITON_SWMM_toolkit/tarball/master>`_::

    curl -OJL https://github.com/lassiterdc/TRITON_SWMM_toolkit/tarball/master

Once you have a copy of the source, you can install it with::

    cd TRITON_SWMM_toolkit
    uv pip install .
